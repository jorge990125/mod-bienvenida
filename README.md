## mod-bienvenida

## Description
Este modulo permite mostrar un menu al iniciar sesión el cual podrá agregar información sobre su servidor

##Requisitos
- AzerothCore v3.0.0+


## Installation

```
1) Simplemente ejecute `git clone` del módulo que se encuentra en el directorio `modules` de su fuente AzerothCore o cópielo y péguelo manualmente.
2) Vuelva a ejecutar cmake y lance una compilación limpia de AzerothCore.
```
